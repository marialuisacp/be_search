O que é este projeto?

Biblexplorer é uma ferramenta online (gratuita) que visa a exploração e o estudo da Bíblia Sagrada. O objetivo deste projeto é estimular o interesse das pessoas na Bíblia e oferecer conteúdo informativo para estudo.

Este trabalho se encontra em fase de desenvolvimento e a previsão de lançamento é no começo de 2017. Porém uma versão simplificada do projeto já se encontra disponível para dispositivos móveis (android e iOS) em fase de testes.

Visualizações da Bíblia Sagrada

Você pode ver a Bíblia de uma forma diferente, através de gráficos e visualizações interativas.

O que este trabalho vai oferecer?

Leitura e descoberta da Bíblia

Ler a Bíblia de uma forma simples, com gráficos e informações sobre os textos e dicionário bíblico para as palavras difíceis.

Visualizações de dados

Um conjunto de conteúdo visual com mapas, gráficos sobre a Bíblia Sagrada.

Pesquisa

Já pensou em uma máquina de busca na Bíblia, onde você digita um termo e a ferramenta te retorna os textos bíblicos relacionados à aquele assunto? É mais ou menos isso.

Be Aplicativo

Uma versão para dispositivos Android e iOS onde você pode ler a Bíblia fazer suas próprias anotações dos textos bíblicas.
